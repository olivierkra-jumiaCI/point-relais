# Jumia Point Relais

Application Svelte pour la gestion et l'affichage des agences et points relais Jumia en Côte d'Ivoire.

## Structure du projet

- `PointRelais.svelte` : Composant principal.
- `src/components/` : Composants UI modulaires.
- `src/lib/data.js` : Données statiques (agences, régions, FAQs).
- `src/index.css` : Styles globaux.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```
